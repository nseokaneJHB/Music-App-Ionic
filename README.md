# **Ionic Music Mobile App**

## Requirements:
- NPM: v6.14.14
- Node: v14.17.5
- Nest CLI: v7.6.0
- Ionic: v6.17.1
- Angular CLI: v10.2.3
- MongoDB Shell: v4.4.3

## Stack used:
- Frontend - Ionic (Agular && TypeScript)
- Backend - NestJS
- Data Store - MongoDB
